# Force Per4mance - Elite Football Academy

Force Per4mance (FP4) is an elite football development program based in Landskrona, Sweden. This repository contains the source code for the official website, player portals, and administrative dashboard.

## Project Structure

- `index.html`: Main landing page.
- `portal-local.html`: Registration portal for local youth players.
- `portal-international.html`: Application portal for international players.
- `style.css`: Global styles and design system.
- `main.js`: Client-side logic and interactions.
- `components/`: (WIP) Modular UI components.
- `docs/`: Project documentation and roadmap.

## Tech Stack

- **Frontend**: HTML5, Vanilla CSS, JavaScript.
- **Backend (Planned)**: Firebase (Authentication, Firestore, Storage).
- **Hosting**: Google Antigravity / Firebase Hosting.

## Getting Started

Simply open `index.html` in a browser to view the site locally.
